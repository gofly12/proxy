# 什么是机场？

机场通常是指提供 Clash、SSR、V2Ray、Trojan 等代理订阅服务的平台。

用户购买套餐后，可以获得专属订阅链接，通过 Clash、Shadowrocket、Stash 等客户端导入节点后使用。

相比传统 VPN，机场通常拥有：

* 更多节点地区
* 更快的连接速度
* 更丰富的线路选择
* 更好的流媒体解锁能力
* 更低的使用成本

因此越来越多用户选择机场作为长期科学上网方案。

# 机场常见问题

## 机场和 VPN 有什么区别？

VPN 通常采用固定服务器模式。

机场则提供多个节点、多个国家线路以及更灵活的订阅配置方式。

## 新手适合哪种机场？

推荐：

* 超实惠加速
* 飞鸟机场
* 极连云

可以先从月付套餐开始体验。

## 为什么机场价格差异这么大？

主要与以下因素有关：

* 是否为专线
* 节点数量
* 带宽成本
* 流媒体支持情况
* 用户规模

因此不能单纯通过价格判断机场质量。
